Game Development I:

Assessment 1: Game Review

You will work in a small group to develop a presentation on an Indie Game title. You will assign roles within your group, negotiating on who is best to cover each role. You should consider forming your team based on including classmates with varied interests within the context of game development. The group will analyze an existing indie game from the perspective of key industry roles - e.g. Game Developer (game structure, mechanics, logic), Level Designer, Game Writer, Concept Artist, Producer (production methods, schedules, marketing, Social Media campaigns, liaising with fans).

You must define what you think the motivations and the goals of each person in these key industry roles were in creating the game.
